HTML
  • HTML was introduced by Tim Berners in 1991.
  • HTML is a Language used to Create Page Structure.
  • HTML stands for Hyper Text Markup Language.
  • The Current version Html is Version 5.
  • hyper Text means the text that refers to another text
TAGS
  • Keywords are written inside the tags.
  • There are three main tags In Html.They are html,head,body tags.
HEADER TAGS
  • heading tags, are used to separate headings and subheadings on a webpag
  • They rank in order of importance, from H1 to H6
  • Header tags improve the readability of a webpage
PARAGRAPH TAG
  • It is used to define paragraphs of text.
  • Its a block level element meaning it starts with new line.
  • They are used to present the text content clearly nad effectively.
FORMATTING TAGS
  1. Bold: Specifies boldness.Only for style purpose
  2. Italic:Give the text in Italic font
  3. Strong:makes the text bold.Only used to mark the importance.
  4. Underline:Underlines the text.
  5. Mark:Marks the text means highlights it.
  6. represents the text that is quoted from the another website.
BLOCK AND INLINE
  • Block level elememts:Always starts in new line and takes the whole block
  • Inline: It will take only required space.
TABLES
  • Tables are used to represent data in tabular format.
  • They are defined with table tag.
  • Tables are divided into rows and columns.
  • Rows are represented with tr and columns rae represented with td and column headings are represented with th tag.
  • There are many features in tables.
  • In table we can give caption,cellpadding,cellspacing,background color,width and height.And also group and merge them.
URL
  • URL stands for Uniform Resource locator is the address of a unique resource on the internet
  • A URL is composed of different parts, some mandatory and others optional.
  • The first part of the URL is the scheme, which indicates the protocol that the browser must use to request the resource
  • After that there is domain.In domain there are Sub domain,Second level domain,Top level domain.
  • Follows the path and parameters are given after that.
