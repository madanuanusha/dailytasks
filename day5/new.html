Features of VS Code

1.Extensions: It has wide rang of extensions to add functionalities like to select themes,language support,tools
2.Debbuging:We can run and deug the code within the editor

Shortcuts
Shortcut Description
Ctrl+s to save
ctrl+c copy
ctrl+v paste
ctrl+x cut
ctrl+A To select All
Alt+up moves the current line or selected line up by one line
Alt+down moves the current line or selected line down by one line
Ctrl+z undo
Ctrl+y redo
Ctrl+f allows you to search for text
Ctrl+Shift+arrows(up/down) allows you to quickly select additional lines without the mouse
Ctrl+Shift+Alt+down arrow for multiple cursors
Ctrl+Shift+N New Window
Ctrl+Shift+W Close Window
